 Dự án Quản lý Sinh viên (Student Manager)
Đây là một ứng dụng quản lý sinh viên đơn giản được xây dựng bằng C# và Windows Forms (.NET Framework / .NET Core).
 Yêu cầu Hệ thống

* **Hệ điều hành:** Windows 10/11
* **Môi trường phát triển:** Visual Studio 2019/2022
* **SDK:** .NET Framework 4.7.2 (hoặc phiên bản .NET mà project của bạn sử dụng).

*  Yêu cầu Hệ thống
* **Hệ điều hành:** Windows 10/11

   Bắt đầu

1.  Clone Repository về máy của bạn:
    ```bash
    git clone [https://github.com/pquoc7522/StudentManager.git](https://github.com/pquoc7522/StudentManager.git)
    ```
2.  Mở Solution file (`StudentManager.sln`) bằng Visual Studio.
3.  Nhấn **Start** (F5) để chạy ứng dụng.
4.  Sử dụng các nút Thêm/Sửa/Xóa để thao tác với dữ liệu sinh viên.
